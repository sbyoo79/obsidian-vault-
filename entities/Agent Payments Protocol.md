# Agent Payments Protocol

**AI 에이전트가 사용자 대신 자율 결제를 수행** 할 때의 규칙·표준. [[Google]] 이 2026 I/O 에서 공개. 약칭 **AP2**.

## 원안 / 출처

- 2026 I/O 컨퍼런스에서 [[Google]] 공개
- Universal Cart (검색·[[Gemini]] 앱 상품 한 곳 결제) 와 함께 발표

## 핵심 원리

- 에이전트가 사용자 의사를 어디까지·어떤 규칙으로 대행할 수 있는지의 표준
- 자율 결제 = "사용자가 명시 승인하지 않은 거래까지 에이전트가 진행" 의 가능성 영역. 그 경계를 표준화하는 시도

## 본 vault 에서의 의미

[[Agentic Engineering]] 의 다음 단계 — 에이전트가 코드·문서 수준을 넘어 **금전 결정** 까지 자율 수행하는 영역. [[Vibe Coding]] 의 "검증 없는 채택" 사상이 결제 영역에 적용되면 **금전 손실** 위험으로 직결. 본 vault 의 "검증 루프" 사상이 금전 영역에서도 필요하다는 신호.

[[ai-market-pulse]] 의 추적 변수 — [[Google]] 풀스택 전략의 핵심 카드 중 하나. 표준 선점이 곧 생태계 주도권.
