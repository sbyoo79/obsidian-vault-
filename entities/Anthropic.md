# Anthropic

미국 AI 안전 중심 회사. **Claude** LLM 시리즈 제조. OpenAI 출신들이 2021 년 설립. Constitutional AI 같은 안전 기법 연구로 차별화.

## 개요

[[OpenAI]] 출신 [[Jack Clark]] 등이 2021 년 설립. **AI 안전** 을 회사 정체성의 중심에 두는 회사로, Constitutional AI, RLAIF, Responsible Scaling Policy 같은 안전 기법으로 차별화한다. 주력 제품은 [[Claude]] 시리즈 (Opus / Sonnet / Haiku) 와 Claude Code (CLI).

2026 년 들어 **AI 가 AI 모델 개발을 가속화** 한다는 방향을 4대 연구 의제 중 하나 ([[AI-driven R&D]]) 로 공식화. 같은 흐름에서 2026-05 [[Andrej Karpathy]] 를 pretraining 팀으로 영입, "Claude 로 Claude 를 더 잘 만드는" 연구 팀을 구성했다. 사내에 800 개 이상의 AI 에이전트를 운영하며 엔지니어 개발 속도가 20~40% 빨라졌다고 보고.

이차 시장 평가 ≈ 1조 달러로 [[OpenAI]] (≈ 8,800억 달러) 를 이미 추월.

## 본 vault 에서의 의미

본인이 매일 사용 중인 [[Claude]] 의 제조사. 본 vault 의 active agent 가 Claude — 즉 Anthropic 의 제품이 vault 의 인프라. 멀티 에이전트 워크플로 ([[Agentic Engineering]]) 의 한 축 ([[Claude]] = 리뷰 / 통합 담당).

[[AI-driven R&D]] 의제는 본 vault 가 추적 중인 [[Self-Improving Agent]] 흐름의 산업 최대 규모 실증 사례. Karpathy 합류는 본인이 vault 에서 추적해 온 인물의 다음 행보가 정확히 이 흐름과 맞물린 사건.
