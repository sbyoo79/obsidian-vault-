# Antigravity

[[Google]] 의 에이전트 통합 코딩 IDE. [[Gemini]] 기반. 2026-05 시점 본인 **미검증**.

## 개요

[[Google]] 이 코딩 워크플로 자체를 에이전트 중심으로 재설계한 도구. 초기 버전은 VS Code 스타일 IDE 였으나, 2.0 에서 "에이전트 중심 통합 도구" 로 강제 전환되며 사용자 반발 발생.

## 핵심 기능

- [[Gemini]] 기반 코딩 에이전트
- [[Google]] 생태계 통합 (인증·실행 워크플로)
- 에이전트 중심 UX (2.0 이후)

## 이 vault 에서의 사용

**현재: 미사용**. 본 vault 의 멀티 에이전트 워크플로는 [[Claude]] Code + [[Codex]] + [[ChatGPT]] 조합 (회사 AGENTS.md).
