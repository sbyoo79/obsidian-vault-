# Atlassian

호주 dev tools / 협업 SaaS 회사. Jira · [[Bitbucket]] · Confluence 등 본인 일상 작업 도구 다수 제공.

## 개요

2002 년 설립. 개발팀 협업 SaaS 의 대표 벤더 — 이슈 추적 (Jira), 코드 호스팅 (Bitbucket), 위키 (Confluence), CI/CD (Bitbucket Pipelines) 의 통합 생태계. 회사 AGENTS.md 의 작업 루틴이 Jira 워크플로 위에서 작동하므로, 본 vault 사용자에게 핵심 인프라 회사.

2026 년 자체 AI 에이전트 **Rovo Dev** 를 dev tools 에 통합. 같은 시기 [[Bitbucket]] Pipelines 가 [[Claude]] Code 를 추가 제공자로 지원 — Atlassian 자체 에이전트 외 외부 LLM 도구 통합으로 확장 시작.

## 본 vault 에서의 의미

본인 회사가 Jira + [[Bitbucket]] 사용 중. 회사 공통 AGENTS.md 의 "JIRA 티켓 생성 → 브랜치 생성 → 본 작업" 루틴이 Atlassian 생태계 위에서 작동. 즉 Atlassian = 본인 회사 워크플로의 기반 인프라.

[[ai-market-pulse]] 추적 변수 — Atlassian 의 [[Bitbucket]] Pipelines 가 [[Claude]] Code 를 지원하기 시작한 사실은 [[Anthropic]] ecosystem expansion 신호. [[Google]] 의 풀스택 통합 ([[Antigravity]]) 과 다른 방식의 침투 — 외부 dev tools 회사가 [[Anthropic]] 제품을 자사 파이프라인에 끼우는 패턴.
