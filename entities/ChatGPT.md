# ChatGPT

[[OpenAI]] 가 만든 LLM 챗 인터페이스. 2022 년 말 공개 이후 LLM 의 대중화를 견인한 대표 제품.

## 개요

GPT 시리즈 모델 (GPT-3.5 → GPT-4 → GPT-4o → 이후 라인업) 을 일반 사용자가 쉽게 쓸 수 있도록 제공하는 챗 형식 서비스. 2022 년 11 월 공개 직후 단기간에 사용자 1 억 명을 돌파하며 LLM 응용 시장의 폭발을 만들었다. 이후 음성, 비전, 코드 인터프리터, GPTs (커스텀 GPT), 메모리, agent 모드 등이 단계적으로 추가됐다.

[[Claude]] 와 함께 LLM 챗 도구의 대표 비교 대상이다. 모델 라인업·툴 통합·요금제는 자주 바뀌므로 한 시점의 사실은 빠르게 낡는다.

## 본 vault 에서의 의미

[[Claude]] 와 함께 LLM 활용 사례에서 자주 등장한다. 본 vault 의 active agent 는 [[Claude]] 지만 외부 source 에서 "AI 에게 물어보자" 라고 하면 대부분 ChatGPT 사례가 등장하므로 backlink hub 로 자주 쓰일 entity.
