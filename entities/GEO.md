# GEO

**Generative Engine Optimization** — AI 검색 엔진([[ChatGPT]] / Perplexity / Google AI Overview / Claude 등) 답변에 자사 콘텐츠가 인용·반영되도록 최적화하는 마케팅 활동.

## 개요

전통 [[SEO]] 의 후속·확장 개념. SEO 가 "검색 결과 페이지 상위 노출" 이었다면, GEO 는 "AI 답변 본문에 인용·언급되기" 가 목표다. 사용자 경험이 검색 결과 클릭 → AI 답변 즉시 소비로 옮겨가면서 등장한 새 분야 (2023~).

측정 지표는 SEO 와 다르다. 순위·클릭 대신 **AI Citation** (브랜드/도메인이 출처로 언급), **AI Visibility** (AI 가 콘텐츠를 발견·이해할 수 있는 상태), **Query Fan-out** (AI 가 프롬프트를 확장해 탐색하는 방식) 같은 새 지표 체계가 형성 중이다.

핵심 함정은 **단일 KPI 추종**. 인용률 단독 KPI 는 콘텐츠 양으로 부풀릴 수 있어 신뢰성이 떨어진다 — [[yozm-geo-leading-indicators]] 의 주된 메시지.

## 본 vault 에서의 의미

본 vault 가 다루는 LLM/agent 활용의 **반대 면**. vault 는 "내가 LLM 을 어떻게 잘 쓰는가" 라면, GEO 는 "내 콘텐츠가 LLM 에게 어떻게 잘 소비되는가". [[Context Engineering]] 의 외부 공급자 측 시각으로 볼 수 있다.

## Key facts

- 명명: **G**enerative **E**ngine **O**ptimization
- 등장: 2023~ AI 검색 (Perplexity, ChatGPT Search, Google AI Overview) 활성화와 함께
- 측정 지표: AI Citation, AI Visibility, Query Fan-out + SEO 전통 지표 (브랜드 검색 유입 등)
- 함정: 단일 KPI (인용률) 추종 → 단기 조작 가능
- 권장: 전조 지표 + 최종 KPI 병행

## 원안 / 출처

특정 개인 원안 없음. 2023~2024 SEO 커뮤니티에서 자연 발생 후 [[Semrush]] 같은 도구 회사가 용어를 정착시키는 흐름.

## 핵심 원리

| 원리 | 설명 |
|------|------|
| 콘텐츠 = 학습 데이터 | AI 가 학습/참조할 수 있도록 구조화된 콘텐츠 |
| 인용 가능성 | 사실 정확성, 정량 데이터, 1 차 출처 명시가 인용 확률 상승 |
| 측정은 다층 | 단일 KPI 가 아닌 전조 지표 + 최종 지표 결합 |
| 데이터 기반 검증 | 방법론 체크리스트보다 실측 / 테스트 우선 |
