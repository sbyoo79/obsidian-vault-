# Gemini

[[Google]] 의 프론티어 LLM 시리즈. [[Claude]] · GPT 와 같은 축의 멀티모달 모델. 2026-05 기준 본인 **미검증** 상태.

## 개요

[[Google]] DeepMind 가 개발한 LLM 시리즈. 라인업은 Pro / Flash / Omni 로 구성 — Pro 는 최상위, Flash 는 경량·고속, Omni 는 영상 생성. 2026 I/O 에서 풀스택 통합 전략의 중심 브랜드로 자리매김.

[[Google]] 검색·지메일·구글 독스·안드로이드 XR 등 [[Google]] 생태계 전반에 [[Gemini]] 가 침투 중. 본인 [[llm-tool-comparison]] 에서는 미검증 항목으로 분류.

## 핵심 기능

- 멀티모달 (텍스트·이미지·영상·음성)
- 도구 선택 능력 업계 최상위 (Terminal-Bench 2.1 기준)
- [[Google]] 생태계 통합 (지메일·독스·검색·결제)
- 에이전트 자율 실행 (Gemini Spark)

## Key facts (2026-05 시점)

- **Gemini 3.5 Flash**:
  - Terminal-Bench 2.1: **76.2%** (vs GPT-5.5 78.2%)
  - 도구 선택·멀티모달 업계 최상위 평가
  - 토큰 가격 **직전 대비 3배 인상** (Pro 대비 25% 저렴)
  - 고객 사례: "다른 최상위 모델 80% 작업을 Flash 로 이전 → 연 $10억+ 절감"
- **Gemini Omni**: 영상 생성. 커뮤니티 평가 "Sora 우위 불명확"
- **Gemini Pro**: 2026 I/O 시점 출시 지연
- **Gemini Spark**: 클라우드 자율 실행 에이전트. 여름에 자율 문자·이메일 + 로컬 브라우저 제어 추가 예정
