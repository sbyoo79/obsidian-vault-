# Google Search Console

Google 이 무료로 제공하는 검색 성과 모니터링·진단 도구. 사이트 소유자가 자기 사이트가 Google 검색에서 어떻게 노출/클릭되는지, 인덱싱 상태가 어떤지 확인한다.

## 개요

이전 명칭은 "Google Webmaster Tools". 2015 년 현재 이름으로 리브랜딩. SEO 측면의 사실상 표준 도구 중 하나이며, GA4 와 함께 가장 자주 묶이는 분석 도구. 사이트 소유 인증 후 사용 (도메인 / URL prefix 두 방식).

[[GEO]] 시대에도 **브랜드 키워드 검색 유입 추이** 같은 [[전조 지표]](페이지 미생성) 측정에 여전히 중요하다 — AI 답변 노출이 실제 사람의 검색 행동으로 이어지는지 추적하는 1 차 데이터 소스.

## 본 vault 에서의 의미

SEO/GEO 영역 source 가 들어올 때 자주 등장하는 표준 도구. backlink hub 로 자주 쓰일 entity.

## Key facts

- 제조: Google
- 가격: 무료
- 데이터 소급 범위: 16 개월
- 주요 측정 항목: 노출(impression), 클릭, CTR, 평균 순위, 인덱싱 상태, Core Web Vitals

## 핵심 기능

- **Performance report** — 검색 노출/클릭/순위 추이
- **URL inspection** — 특정 URL 의 인덱싱 상태 확인
- **Sitemap 제출**
- **Coverage / Indexing** 진단
- **Core Web Vitals** — 페이지 속도/UX 지표
- **Manual actions** — 구글 패널티 알림
