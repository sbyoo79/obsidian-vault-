# IntelliJ

JetBrains 의 Java/Kotlin 중심 IDE. 본인 일상 주력 IDE. **LLM 도구 시대에도 막강한 도구로 위치 유지**.

## 개요

리팩터링·코드 분석·디버깅·테스트 실행 등 IDE 본연의 기능에서 업계 최상위. Spring · Gradle · Database 통합 지원이 본인 [[Java]] 작업의 베이스. 본 vault [[self-profile]] 의 도구 스택에 명시되어 있는 핵심 도구.

## 핵심 기능

- 강력한 리팩터링 (rename / extract / inline / move)
- 정적 분석 + 인텐션 액션
- Spring · Gradle · MyBatis · JPA / Hibernate 통합
- DB 도구 (DataGrip 수준)
- 디버거 / 프로파일러
- 다중 언어 (Java / Kotlin / Python / SQL / JS / etc)

## 이 vault 에서의 사용

본인의 IDE 측 핵심 도구. [[Claude]] Code / [[Codex]] 같은 LLM 도구는 **다른 레이어** — IDE 가 코드 편집·실행·디버깅의 베이스고, LLM 도구는 그 위에서 코드 생성·리뷰·문서화를 보조한다.

[[llm-tool-comparison]] 에서 "도구 풀의 다른 축: IDE" 절로 명시 — LLM 도구 비교가 본인 전체 dev stack 의 일부 레이어임을 잊지 않기 위함.

## Key facts

- 본인 주력 IDE, 21 년차 백엔드 작업의 베이스
- LLM 도구 시대에도 막강함 — "코드 리팩터링·구조 변경" 영역에서 LLM 보다 빠르고 안정적인 경우 많음 (본인 평가)
- 라이센스: 상업용 (Ultimate)
