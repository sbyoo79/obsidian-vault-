# Obsidian

local markdown 파일 기반 PKM 앱. `[[wiki link]]` 와 graph view 가 핵심. plugin 생태계로 확장 가능.

## 본 vault 에서의 역할

[[LLM Wiki]] 패턴의 browser / editor 역할. Claude Code 가 file system 으로 직접 read/write, Obsidian 은 사용자가 vault 를 탐색·편집하는 GUI.

## 이 vault 설치 plugin

- `obsidian-git` — git 동기화 plugin (현재 비활성 — WSL git 으로 직접 동기화)
- `mermaid-tools` — mermaid diagram
