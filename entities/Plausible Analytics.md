# Plausible Analytics

프라이버시 우선의 가벼운 웹 분석 도구. **쿠키리스 / GDPR 친화 / 스크립트 < 1 KB**. self-host (AGPL 오픈소스) 또는 cloud (유료 약 9 USD/mo~).

## 개요

GA4 의 무거움 / 개인정보 추적 부담 / 쿠키 배너 필요성에 대한 대안. **단순한 트래픽 / 채널 / 페이지 인사이트** 만 제공 — 세션 리플레이 / 퍼널 같은 깊은 분석은 X.

## 본 vault 에서의 의미

[[yozm-analytics-for-vibe-coders]] 에서 콘텐츠 / 마케팅 사이트의 GA4 대안으로 추천. 본인 작업에서 가벼운 인사이트만 필요한 영역 (예: 회사 블로그 / 랜딩) 의 후보.

## 핵심 기능

- 페이지뷰 / 방문자 / 채널 / 디바이스 / 국가 등 기본 인사이트
- Cookieless — GDPR / CCPA / PECR 친화
- 스크립트 크기 < 1 KB
- self-host (AGPL) 또는 cloud

## Key facts

- 오픈소스 (AGPL)
- Cloud 플랜: 약 9 USD/월~ (트래픽 기반)
- self-host 무료
- 매우 가벼움 — 스크립트 / DB / 리소스 모두 최소
