# SEO

**Search Engine Optimization** — 검색 엔진(Google / Naver / Bing 등) 결과 페이지에서 자사 콘텐츠가 상위에 노출되도록 최적화하는 활동.

## 개요

1990 년대 후반 검색엔진 등장과 함께 시작된 디지털 마케팅의 핵심 분야. 알고리즘 진화에 맞춰 키워드 매칭 시절 → 링크 분석 (PageRank) → 콘텐츠 품질·UX → E-E-A-T (Experience/Expertise/Authoritativeness/Trustworthiness) 같은 패러다임으로 변해왔다.

2023~ AI 검색 등장으로 그 후속·확장 영역인 [[GEO]] 와 공존 시기에 진입했다. 둘은 대체 관계가 아니라 **레이어 차이** — SEO 가 "검색 결과 페이지", GEO 가 "AI 답변 본문" 이라는 다른 surface 를 최적화 대상으로 잡는다.

## 본 vault 에서의 의미

GEO 와 비교 컨텍스트로 자주 참조될 entity. 본 vault 의 LLM 활용 영역과 직접 맞닿지는 않지만, "AI 가 콘텐츠를 어떻게 소비하느냐" 라는 큰 흐름의 출발점.

## Key facts

- 시작: 1990 년대 후반
- 주요 지표: 노출, 클릭률, 순위, 트래픽
- 표준 도구: [[Google Search Console]], GA4, Ahrefs, Semrush, Naver Search Advisor
- 알고리즘 진화: 키워드 → PageRank → 콘텐츠 품질 → E-E-A-T → AI 통합

## 핵심 원리

- 검색 의도 파악 (search intent)
- 콘텐츠 품질 + 신뢰성
- 기술 SEO (속도, 모바일, 구조화 데이터)
- 권위·신뢰 (백링크, 도메인 권위)
