# Self-Distancing

자신의 상황을 3 인칭처럼 바라봄으로써 더 현명한 판단·정서 조절을 가능하게 하는 인지 기법.

## 개요

워털루대 [[Igor Grossmann]] 과 미시간대 Ethan Kross 등이 실증적으로 정리한 개념. 자기 문제를 "내가 ~ 하고 있다" 가 아니라 "그가 / 그녀가 ~ 하고 있다" 로 재기술하면, 감정 강도가 줄고 더 현명한 판단이 나온다. [[Solomon's Paradox]] 의 직접적인 완화책.

언어적 재기술 외에도 글쓰기/일기, 시간 거리두기 ("10 년 뒤 나라면"), 공간 거리두기, 그리고 최근에는 **LLM 을 통한 역할 반전** 까지 여러 도구 형태로 활용된다.

## 본 vault 에서의 의미

vault 의 자기 ingest 자체가 self-distancing 의 한 형태 — 머릿속 생각을 markdown 페이지로 외부화하면 자연스럽게 거리가 생긴다. discussion source 를 사후에 다시 읽을 때 특히 효과적이다.

## Key facts

- 핵심 메커니즘: 1 인칭 → 3 인칭 (또는 외부 관찰자) 시점 전환
- 효과: 감정 강도↓, 분석적 사고↑, 의사결정 질↑
- 적용 도구: 일기/저널링, 친구/코치 대화, 시간 거리두기, LLM 역할 반전

## 핵심 원리

- **언어 전환** — "I → he/she/your name"
- **시점 전환** — 자기 → 외부 관찰자
- **시간 전환** — 지금 → 미래·과거 시점
- **공간 전환** — 자기 안 → 자기 밖
