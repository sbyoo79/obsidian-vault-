# Simon Willison

Django 공동창립자. LLM 활용·툴·웹 엔지니어링 영역에서 영향력 있는 글을 쓰는 엔지니어·블로거.

## 개요

Django 웹 프레임워크 공동창립자 (2005, 1990 년대 후반부터 활동). 이후 datasette (오픈소스 데이터 탐색 도구), llm CLI 등 오픈소스 프로젝트와 함께, simonwillison.net 블로그로 LLM 시대의 실전 활용·관찰을 빠르게 정리해 공유한다.

LLM 영역에서 자주 인용되는 이유는 **slogans 가 아닌 정확한 정의** 를 제시하기 때문. [[Vibe Coding]] 에 대해서도 핵심을 다시 박았다 — "AI 가 코드를 썼느냐" 가 아니라 "**검증을 했느냐**" 가 본질.

## 본 vault 에서의 의미

본 vault 가 추구하는 "정확한 정의 + 외부화" 사상과 같은 결의 활동을 하는 인물. 그의 블로그는 향후 source 가 자주 추가될 후보.
