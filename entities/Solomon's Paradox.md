# Solomon's Paradox

타인의 문제에는 현명하게 조언하지만 자기 문제에는 그러지 못하는 인지 편향.

## 개요

이름은 구약의 솔로몬 왕 일화에서 따왔다 — 백성과 외부 분쟁에는 지혜로웠지만 본인 가정 문제(왕위 계승, 자식들 갈등) 는 잘 처리하지 못했다는 모순. 심리학에서는 워털루대 [[Igor Grossmann]] 가 2014 년 즈음 실증 연구로 명명·정리했다.

원인은 단순하다: 자기 문제는 감정과 맥락 압박으로 시야가 좁아진다. 타인의 문제는 한 발 떨어진 위치에서 보기에 분석이 잘 된다. 거리감 자체가 인지 자원의 차이를 만든다.

## 본 vault 에서의 의미

LLM 을 단순 자동화 도구가 아닌 **거리두기 보조 장치** 로 쓰는 발상의 출발점. [[Self-Distancing]] 과 [[Role-Reversal Dialogue]] 가 직접 연결된다. 본 vault 의 자기 ingest 도 약한 형태의 self-distancing — 머릿속 생각을 위키로 외부화하면 거리감이 생긴다.

## Key facts

- 명명 유래: 구약 솔로몬 왕 일화
- 실증 연구: [[Igor Grossmann]] 등 (2014~)
- 핵심 변수: **자기 vs 타인** 거리감
- 완화책: 의도적 거리두기 ([[Self-Distancing]])

## 원안 / 출처

- Grossmann, I., & Kross, E. (2014) — "Exploring Solomon's paradox: Self-distancing eliminates the self-other asymmetry in wise reasoning about close relationships"

## 핵심 원리

| 관찰 | 의미 |
|------|------|
| 타인 시점 → 분석적 | 감정 압박이 약하다 |
| 자기 시점 → 즉시 반응 | 감정·맥락이 추론을 좁힌다 |
| 거리감 = 자원 | 시점을 인위적으로 옮기면 추론이 회복된다 |
