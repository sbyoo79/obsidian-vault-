# Umami

가벼운 오픈소스 웹 분석 도구. **self-host 완전 무료**, GDPR / 프라이버시 친화 (쿠키리스, 비개인화). [[PostHog]] 보다 단순하지만 그만큼 운영 부담 낮음.

## 개요

이벤트 / 페이지뷰 추적 + 기본 대시보드. PostHog 같은 통합형 (세션 리플레이 / 피처 플래그 / 설문) 은 X, 순수 web analytics 만. 데이터 오너십 / 단순함 / 비용 0 이 핵심 가치.

## 본 vault 에서의 의미

[[yozm-analytics-for-vibe-coders]] 에서 PostHog 의 가벼운 대안으로 소개된 도구. 본인 광고 플랫폼이나 사이드 프로젝트의 self-host 분석 후보 — [[product-analytics-for-self-built-systems]] 노트 참조.

## 핵심 기능

- 페이지뷰 / 이벤트 추적
- 기본 대시보드 (acquisition / engagement / retention)
- 쿠키리스 / GDPR 친화
- self-host (Node.js + PostgreSQL / MySQL)

## Key facts

- 오픈소스 (MIT)
- 완전 self-host 무료
- 무료 cloud 플랜도 있음 (제한 적음)
- 프라이버시 우선 — 개인정보 수집 최소화
