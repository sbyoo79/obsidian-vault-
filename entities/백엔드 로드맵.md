# 백엔드 로드맵

백엔드 개발자가 갖춰야 할 기술 영역을 단계·계층 별로 정리한 학습 가이드. 진입장벽이 높아진 현대 백엔드 환경에서 "무엇부터 배워야 하는가" 의 혼란을 해소하는 자료 패턴.

## 개요

자바·JSP·SQL 만으로 충분했던 과거와 달리 현대 백엔드는 운영체제·네트워크·웹/WAS·프레임워크·아키텍처·언어·DB·클라우드·DevOps 까지 광범위한 영역을 다룬다. 로드맵은 이 영역들을 학습 우선순위로 배열해 입문자가 길을 잡을 수 있게 한다.

## 본 vault 에서의 의미

본인 [[self-profile]] 의 기술 스택 5 단계 자기 평가와 직접 매핑되는 분야. **주니어·후배에게 추천하는 학습 자료의 종류** 로 인지하고, [[backend-roadmap-for-juniors]] 노트에 시니어 관점 보강을 정리.

## Key facts

- 다루는 영역: OS / 네트워크 / 웹·WAS / 프레임워크 / 아키텍처 / 언어 / DB / 클라우드 / DevOps
- 우선순위: 일반적으로 하위 layer (OS / 네트워크) → 상위 추상화 (프레임워크 / 아키텍처) → 응용 (DB / 클라우드 / DevOps)
- 한계: 단순 기술 나열로는 실제 운영 / 마이그레이션 / 리더십 / AI 활용 같은 시니어 영역까지 커버 X
