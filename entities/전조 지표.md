# 전조 지표 (Leading Indicator)

최종 KPI 보다 **먼저 움직이는** 측정 지표. 최종 결과를 기다리지 않고 가설·전략의 방향성을 일찍 판정할 수 있게 해주는 신호.

## 개요

비즈니스·제품·마케팅의 최종 KPI (매출 / PMF / AI 인용 등) 는 시간 차가 크고 신호 강도가 늦다. 전조 지표는 그 최종 결과로 가는 **흐름의 초기 단계** 를 측정해 가설을 빨리 검증할 수 있게 한다.

예:

| 분야 | 최종 KPI | 전조 지표 |
|------|---------|----------|
| 이커머스 | 매출 | 카트 추가 / 결제 시작 / 첫 구매까지 시간 |
| GEO | AI 인용 비율 | 브랜드 키워드 검색 / GSC impression / indexable 비율 |
| 제품 (PMF) | PMF 달성 | 활성화 비율 / 재방문 비율 / [[Aha Moment]] 도달률 |
| AI 코딩 도구 | 산출 코드 품질 | 채택률 / 수정 시간 / 롤백률 (cf. [[Agentic Engineering]]) |

## 본 vault 에서의 의미

여러 영역에서 동일 사상으로 등장하는 **메타 개념**. 단일 KPI 매몰을 회피하는 측정 원칙으로 [[GEO]] · [[Product Analytics]] · [[Agentic Engineering]] 모두와 연결.

본 vault 의 `log.md` 자체도 일종의 전조 지표 — 매일 ingest 수 / 새 entity 수 / cross-ref 수가 vault 건강도의 leading indicator. ([[yozm-geo-leading-indicators]] 의 본 vault 와의 연결 절 참조)

## Key facts

- 최종 KPI 의 인과적 선행 신호 (causally upstream)
- 단일 KPI 매몰 회피 — 전조 + 최종 병행이 원칙
- 분야별 정의 다양 (마케팅 / 제품 / DevOps / 엔지니어링)
- 시간 차 단축 → 가설 검증 사이클 빨라짐
