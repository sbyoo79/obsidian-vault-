# 안드레 카파시는, 왜 지금 엔트로픽에 갔나

[[트파원]] 의 [[YozmIT]] 기고. [[Andrej Karpathy]] 가 [[Anthropic]] 에 합류한 사건을 단순한 인재 이동이 아닌, **"모델 사이클 자체가 흔들리는 변곡점"** 신호로 읽는 분석 기사. Karpathy 본인의 X 발표 + [[Anthropic]] 의 'AI-driven R&D' 의제 + [[Jack Clark]] 발언을 묶어 해석한다.

## 핵심 사실

- Karpathy 가 X 로 [[Anthropic]] 합류 발표. 24시간 내 2,500만 조회.
- 합류 메시지: **"다음 몇 년이 LLM 프론티어에서 결정적인 시기"**.
- 본인이 2024-07 설립한 [[Eureka Labs]] (AI 교육 스타트업) 는 사실상 보류. 합류 트윗에 "교육에 대한 열정은 여전, 시간 되면 다시 시작" 언급.

## 맡은 역할 — 사전학습 팀

- LLM 개발에서 가장 비용·시간 큰 단계인 **pretraining** 팀 합류.
- 새 팀 구성, 핵심 미션: **"[[Claude]] 를 이용해 [[Claude]] 의 사전학습 연구를 가속화"**.
- 즉 모델 개발 방법론 자체를 AI 로 푸는 시도. 기존 6개월~1년 모델 사이클을 흔들 가능성.

## Anthropic 의 전략 맥락

[[Jack Clark]] (공동창업자, 2026-05-07 Axios 인터뷰):

> 2028년 말까지 AI 가 자기 후속 모델을 완전히 자율적으로 훈련할 확률이 60% 이상이다.

같은 날 [[Anthropic]] 은 **'[[AI-driven R&D]]'** 를 4대 연구 의제 중 하나로 공식 발표. Karpathy 합류는 이 의제의 인력 보강 신호.

[[Anthropic]] 사내 현황:
- 사내 운영 AI 에이전트 800개 이상
- 엔지니어 개발 속도 20~40% 증가
- 일부 코드는 에이전트에 위임

## Karpathy 의 AI 관점 (인용)

**에이전트 시대 회의론** (2024 인터뷰 인용):

> 모델들은 아직 거기까지 못 왔어요. 업계가 너무 큰 점프를 시도하면서 가장하는데, 사실 그렇지 않아요.

**LLM 본질 비유**:

> 우리는 동물을 만드는 게 아니라 유령이나 영혼을 만들고 있어요. 진화로 훈련되는 게 아니라 인간이 인터넷에 올린 데이터를 모방하도록 훈련되니까요.

→ "인간처럼 보이지만 완전히 다른 종류의 지능" 으로 인식. AGI 도달까지 **약 10년** 필요하다는 입장.

## 합류의 정합성

- Karpathy 의 인식론 = "모델 내부 메커니즘을 더 깊이 이해해야 한다"
- [[Anthropic]] 의 노선 = "더 안전한 AI" = 모델 내부 이해 강조
- 두 입장이 자연스럽게 부합 → 단순한 인재 이동이 아닌 사상적 합치.

## 시장 위치

- [[Anthropic]] 이차 시장 평가 ≈ 1조 달러
- [[OpenAI]] ≈ 8,800억 달러
- 평가 기준 이미 역전.

## 한국 IT 업계 시사점 (저자 주장)

- 기존 **6개월~1년 모델 사이클** 자체가 흔들릴 가능성
- 모델 사이클 + 모델 특성 변화가 동시에 올 수 있으므로 로드맵 기준 재계산 필요

## 본 vault 와의 연결

- [[Andrej Karpathy]] 페이지의 `affiliation` 갱신 필요: [[Eureka Labs]] → [[Anthropic]]. [[Eureka Labs]] 는 "보류" 상태로 명시.
- [[Anthropic]] 페이지의 "개요" 가 비어 있었는데 본 ingest 로 의미·전략 정보 보강.
- [[Vibe Coding]] 명명자였던 Karpathy 가 동시에 [[LLM Wiki]] 도 제안 — 본 vault 가 채택한 "외부화·검증" 사상. 이번 [[Anthropic]] 합류는 그 인물의 다음 행보 → vault 의 주요 인물 추적 대상.
- [[Self-Improving Agent]] / [[AI-driven R&D]] 와 직접 연결. "AI 가 자기를 개선" 의 가장 큰 산업적 실증 시도가 [[Anthropic]] 의 이 방향.
